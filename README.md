## 运行方式

### 1、安装前台项目依赖,进入项目根目录输入


```
cnpm install
```
### 2、运行前台


```
cnpm run dev
```
### 3、安装后台项目依赖,进入server文件夹


```
cnpm install
```
### 4、运行后台项目


```
node server
```
### 5、打开浏览器输入


```
localhost:8080
```
